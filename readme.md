How to use:
- Clone the repo; cd to the script "tls_cipher_info.sh".
- Make it executable: chmod +x tls_cipher_info.sh.
- Run the script, providing the host as the first parameter 
```
 ./tls_cipher_info.sh aws.com
 ```
